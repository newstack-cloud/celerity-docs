# Bluelink Signature v1

Bluelink Signature v1 can be used to authenticate with Celerity components that provide HTTP APIs such as the as the [Workflow Runtime](/workflow-runtime/docs/intro).

You can find the specification for Bluelink Signature v1 in the [Bluelink project documentation](https://bluelink.dev/docs/auth/signature-v1).
