The datastore package in the Celerity Python SDK provides a cloud-agnostic interface for interacting with NoSQL data stores (Amazon DynamoDB, Google Cloud Firestore, Azure Cosmos DB).

For an overview of datastore operations, see the [Celerity Datastore resource type](/docs/framework/applications/resources/celerity-datastore#sdk-operations).

For schema management, type generation and schema contracts, see [NoSQL Datastore Schema Management](/docs/framework/applications/nosql-datastore-schema-management).
