The cache package in the Celerity Python SDK provides a cache abstraction covering all core Redis data structures and a credentials provider for direct Redis client access.

For an overview of cache operations, see the [Celerity Cache resource type](/docs/framework/applications/resources/celerity-cache#sdk-operations).
