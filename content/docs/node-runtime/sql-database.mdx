The SQL database package in the Celerity Node.js SDK provides a credentials provider and an optional connection factory for SQL database access.

For an overview of SQL database operations, see the [Celerity SQL Database resource type](/docs/framework/applications/resources/celerity-sql-database#sdk-operations).

For schema management, type generation and schema contracts, see [SQL Database Schema Management](/docs/framework/applications/sql-database-schema-management).
