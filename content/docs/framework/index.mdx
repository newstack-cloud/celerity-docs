Let's get you up and running with **Celerity in less than 5 minutes**.

## Install Celerity

<InstallCelerity matchPlatform="mac">
To install Celerity on macOS, run the following command:

```bash
curl --proto '=https' --tlsv1.2 -sSfL https://manager-sh.celerityframework.io | sh
```
</InstallCelerity>
<InstallCelerity matchPlatform="linux">
To install Celerity on Linux, run the following command:

```bash
curl --proto '=https' --tlsv1.2 -sSfL https://manager-sh.celerityframework.io | sh
```
</InstallCelerity>
<InstallCelerity matchPlatform="windows">
To install Celerity on Windows, download and run the MSI installer:

<WindowsInstallerDownload />

Or run the following command in Windows Subsystem for Linux (WSL):

```bash
curl --proto '=https' --tlsv1.2 -sSfL https://manager-sh.celerityframework.io | sh
```

</InstallCelerity>
<InstallCelerity fallback>
To install Celerity on macOS, Linux or another Unix-like OS, run the following command in your terminal:

```bash
curl --proto '=https' --tlsv1.2 -sSfL https://manager-sh.celerityframework.io | sh
```
</InstallCelerity>

Using a different system or want to install another way? Check out other ways to install Celerity for your platform [here](/docs/framework/installing-celerity).

This will run a shell script to install the [`bluelink-manager`](https://github.com/newstack-cloud/bluelink/tree/main/tools/bluelink-manager) tool with the Celerity profile, which will then install the [Celerity CLI](/docs/cli/getting-started), [Bluelink Deploy Engine](https://bluelink.dev/docs/deploy-engine), a set of core Deploy Engine plugins and the [Blueprint Language Server](https://github.com/newstack-cloud/bluelink/tree/main/tools/blueprint-ls) on your system.
As Celerity consists of multiple software components, the `bluelink-manager` tool makes it easier to install them and manage updates.

## Generate a new project

Generate a new Celerity application using the **simple API template**:

```shell
celerity init --template simple-api my-app
```

You will be prompted to select a programming language for your application, before the project can be generated. Celerity provides template projects for languages with official SDK support, including:

- Python
- Node.js (TypeScript)

See the [runtime](docs/framework/runtime#supported-languages) documentation for more information and to see how you can implement Celerity applications in other languages.

## Start your API locally

Run the development server:

```shell
cd my-app

celerity dev run
```

The API will now be available at `http://localhost:3005`.
You can test the simple API users endpoint with a `GET` request to `http://localhost:3005/users`.

Edit some source code for the users endpoint: the API will **reload automatically** and you can check by re-running `GET /users` request.
